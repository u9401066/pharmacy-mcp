# Project Brief

## 🎯 專案目的

建立一個完整的藥品資訊 MCP Server，讓 AI 助手（如 Claude）能夠：
1. 查詢藥品資訊
2. 計算劑量
3. 檢查藥物交互作用
4. 識別食品藥物衝突
5. 🇹🇼 台灣健保給付查詢與藥品翻譯

## 👥 目標用戶

- 醫療專業人員需要快速查詢藥品資訊
- 藥師需要檢查處方交互作用
- AI 開發者需要藥品功能整合
- 台灣醫療人員需要本地化健保資訊

## 🏆 成功指標

- [x] 專案能透過 MCP 連接 Claude
- [x] 5 個主要功能模組完成
- [x] API 整合穩定運作
- [x] 台灣健保整合完成 (v0.8.0)
- [ ] 測試覆蓋率 > 70%

| 指標 | 狀態 |
|------|------|

## 🚫 範圍限制

- 不提供診斷建議
- 不儲存個人健康資訊
- 不取代專業醫療判斷
- 免費 API 限制（可能需要付費升級）

## 📝 備註

- 所有藥品資訊需附帶免責聲明
- 優先使用官方資料來源（FDA, NIH, TFDA, NHI）
- 台灣健保藥品資料已整合 ✅
